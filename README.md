# 2FA 桌面工具

一个基于 .NET Framework 3.5 和 Windows Forms 开发的双重身份验证 (2FA/TOTP) 桌面端管理工具。

## 功能特性

- **TOTP 验证码生成**：根据时间算法自动生成 6 位动态验证码，支持进度条倒计时显示。
- **屏幕扫码导入**：支持直接在屏幕上框选并识别二维码以导入 TOTP 秘钥（基于 ZXing 库）。
- **账户管理**：支持手动添加、编辑、删除和排序 2FA 账户。
- **安全存储**：账户秘钥及设置支持本地加密存储。

## 运行环境

- Windows 操作系统
- .NET Framework 3.5 或更高版本

## 构建与运行

1. 本项目提供了自动化构建脚本，可直接在项目根目录下双击运行 `build.bat` 进行编译。
2. 编译成功后，可在生成目录中运行 `2FA.exe`。
